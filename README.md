# toric-scope

An online toric variety visualizer for exploring algebraic geometry.

## About

Toric varieties are geometric objects in algebraic geometry described using combinatorial data such as fans and polytopes. This tool provides interactive visualization to help understand these mathematical structures.

## Tech Stack

- **Frontend**: React
- **Backend**: Python
